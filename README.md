# 🌐 Geo API Frontend Dashboard

A modern frontend dashboard for the India Village Geo API.

---

## 🚀 Features

- 🔐 Login system (JWT authentication)
- 🌍 State → District → Subdistrict → Village selection
- 🔍 Search with highlighting & pagination
- 📊 Analytics dashboard with charts
- 🗺️ Google Maps integration

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js
